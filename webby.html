jWebby - A better DOM API

Download

Crawling a website is our territory! And why not use JTidy to clean things up on the fly?

This is a better Java DOM API than the built in one. If a website is broken, why not make sure it works anyways? By default, we fix the sites with JTidy. If this breaks something or if you don't want to install the JTidy library, you can disable it via Webby.setTidy(false);
