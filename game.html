j2D - 2D Java Game Library

Download

Make games without writing that gosh darn game loop!

j2D will allow you to create games with the tick and render functions ready to go.
Capture keys, mouse events and more the best way possible while keeping it simple!
You also have screens ready to go. Instead of using a stage system - Just create different screens!
